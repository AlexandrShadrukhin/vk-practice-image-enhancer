# VK Practice — Image Enhancement (Browser ML)

Веб-приложение для автоматического улучшения изображений прямо в браузере пользователя с использованием TensorFlow.js.

## Демо

[Открыть проект в Vercel](https://vk-practice-image-enhancer.vercel.app/)

## О проекте

Проект реализует систему улучшения изображений в реальном времени без серверной обработки.  
Пользователь загружает изображение, после чего приложение:

- создает задачу обработки;
- выполняет обработку асинхронно;
- отображает прогресс выполнения;
- подбирает параметры коррекции с помощью ML-модели;
- показывает результат и позволяет скачать обработанное изображение.

Коррекция выполняется по следующим параметрам:

- яркость;
- контрастность;
- цветность.

## Основные возможности

- работа полностью на стороне клиента;
- асинхронная обработка без блокировки интерфейса;
- использование Web Worker для вынесения вычислений из основного потока;
- поддержка форматов JPG, PNG, BMP, HEIC;
- отображение статуса задачи и прогресса выполнения;
- предпросмотр исходного и обработанного изображения;
- режим сравнения **до / после**;
- отображение параметров, выбранных моделью;
- скачивание результата обработки.

## Архитектура проекта

Проект состоит из нескольких основных частей:

- **UI-слой** — загрузка изображения, управление задачей, отображение результата;
- **Task Manager** — управление жизненным циклом задач обработки;
- **Worker-слой** — выполнение обработки в фоновом потоке;
- **Image Engine** — вычисление метрик изображения и применение улучшения;
- **ML-слой** — предсказание параметров коррекции с помощью TensorFlow.js.

## Принцип работы

1. Пользователь загружает изображение.
2. Создается задача обработки.
3. Изображение передается в Web Worker.
4. Для изображения вычисляются базовые метрики.
5. ML-модель предсказывает параметры улучшения.
6. К изображению применяются коррекции яркости, контрастности и цветности.
7. Готовый результат возвращается в интерфейс.
8. Пользователь может сравнить результат с исходным изображением и скачать обработанную версию.

## ML-подход

Для проекта была обучена компактная регрессионная модель, которая получает на вход численные метрики изображения и предсказывает параметры коррекции.

### Входные признаки
- средняя яркость изображения;
- показатель контрастности;
- средняя насыщенность.

### Выход модели
- brightness;
- contrast;
- saturation.

Обучение выполнялось отдельно в Python, после чего веса модели были экспортированы и подключены в браузерное приложение через TensorFlow.js.
