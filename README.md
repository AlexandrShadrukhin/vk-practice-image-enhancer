# VK Practice — Image Enhancement (Browser ML)

Система улучшения изображений в реальном времени с использованием ML-модели, работающей прямо в браузере пользователя.

## О проекте
Проект реализует коррекцию:
- яркости
- контрастности
- цветности

Обработка выполняется на стороне клиента (client-side) без нагрузки на сервер.

## Особенности
- Работа во всех современных браузерах
- Асинхронная обработка изображений
- Поддержка JPG, PNG, HEIC, BMP
- Обработка изображений до 15 Мп
- ML-модель подбирает оптимальные параметры улучшения

## Архитектура (в разработке)
- Frontend (React / Web)
- ML-модуль (TensorFlow.js / ONNX / WebAssembly)
- API-логика задач (status / progress / cancel)
